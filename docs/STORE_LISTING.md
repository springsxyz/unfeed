## Name
UnFeed

## Summary (132 chars max)
Remove social media feeds. Keep search, messages, profiles, and links. Use the app — not the infinite scroll.

## Description

```
UnFeed removes the feed so you can use social apps as tools.

THE PROBLEM
Every network opens on an infinite stream designed to keep you scrolling. Sometimes you only wanted to search, reply to a message, or open a link.

WHAT UNFEED DOES
When enabled, UnFeed hides:
• Home feeds
• For You / Explore / Discover
• Recommendation rails and related suggestions (where supported)

You can still:
• Search
• Open profiles
• Use messages / inbox
• Open direct links to posts and videos
• Post and engage on pages you intentionally opened

SUPPORTED SITES
YouTube, Instagram, Facebook, X, Reddit, LinkedIn, TikTok, Pinterest, Substack, Threads, Bluesky

FREE VS PRO
• Free: enable any 3 sites
• Pro: unlock all sites with a one-time purchase (coming soon / via license key)

PERMISSIONS
UnFeed only asks for storage (to save your toggles) and access to the listed social sites so it can hide feeds there. Everything runs locally in your browser.

UnFeed is not affiliated with YouTube, Meta, X, Reddit, LinkedIn, TikTok, Pinterest, Substack, or Bluesky.
```
